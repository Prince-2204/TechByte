Introducing SANDIGDH

(Software for Analysis of Dark Pattern through Intelligent Deep Learning Hypothesis)

It is a chrome-based, machine learning empowered browser extension software with the purpose of detection of
dark patterns in modern day e-commerce websites. It tackles this problem with a multi-stage approach involving
URL- detection,spam/dark pattern text recognition through state-of-the-art machine learning algorithms which
have been pretrained on wide variety of training data and tested in real time. It provides a quantitative analysis,
giving a percentage score detector for dark patterns (ranging 0-100, score directly proportional to the amount
of dark patterns found on the ecommerce website page).
